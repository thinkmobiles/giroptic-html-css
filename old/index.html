compact and rugged:

- 6,9cm x 6,9cm

- 180g / 6,3oz

- Waterproof (IPX 8)

- Magensium alloy and rubber boby

- Colored LED display

Custom optics

3 x 185° ultra wideangle optics

- Ultra-sharp 6 elements glass lens

- Fixed f/2.8 apeture

- Total FOV 360° x 300°

Available modes

- Video (25/30 fps)

- Photo (self-timer / HDR)

- Burst (up to 3 fps)

- Timelapse (5 to 60sd or custom)

- Live (over built-in WiFi or POE base)

Stitching and format

- Unique live stitching technology

- Video: 2k @ 30fps / h.264 - mp4

- Photo: 4k / jpeg

Advanced video capture settings

- 3 microphones for 3D audio

- Gyroscope + accelerometer for image stabilization

Connectivity

- Micro SD recording | Class 10 up to 128GB

- Micro USB (Charging / File transfer)

- WiFi | 2.4GHz / 802.11 b/g/n

- GPS Geo-tagging

- Remote controlling over WiFi

Regular base

- Micro USB port

- Rechargeable and removable lithium-ion battery (1180mAH, 3,7V) (Approximate continuous recording time expected video recording @25fps using a fully charged battery: 60min)

- Regular ¼” tripod and accessories mount

POE base

- Professional Powered Over Ethernet only compatible base

- Mini jack audio input

- RTSP streaming protocol
